# TravelBIG
 -20,30대 젊은층 여행객들을 위한 여행커뮤니티

 -지역별, 여행테마(저가형,먹방,관광명소)등 다양한 분류의 가이드와 정보공유를 통해
  처음 국.내외 여행을 계획하는 이들에게 도움이 되길 원함.

 -테마에 따른 페이지 구성변화로 각종 분야에 커뮤니티로 사용가능(패션,만화,영화,게임 등)
